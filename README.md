# Assignment1_PP
TF-IDF implementation from scratch

This project implements TF-IDF feature extraction without using NLP libraries.

Steps:
1. Text preprocessing
2. Vocabulary construction
3. Term Frequency calculation
4. Inverse Document Frequency
5. TF-IDF matrix generation

Dataset: Indonesian text about Universitas Gadjah Mada.
https://id.wikipedia.org/wiki/Universitas_Gadjah_Mada
